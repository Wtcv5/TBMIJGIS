# 《Geometry-Constrained Rock–TBM Spatial Interaction Graphs》最终修改意见与修改方案

> 适用对象：当前稿件 `main.pdf`（题目：*Geometry-Constrained Rock–TBM Spatial Interaction Graphs for Chainage-Resolved Diagnostic Analysis of Excavation Responses*）。  
> 当前定位：**几何约束岩–机空间交互图 + 部件—里程空间交互描述子 + persistence residual 响应一致性诊断**。  
> 核心原则：**保留图模型主线，放弃 GNN–GRU 作为主模型，不再回到深度预测论文路线。**

---

## 0. 总体判断

当前稿件已经完成了关键转向：从“小样本 GNN–GRU 预测论文”转为“空间交互图诊断论文”。这一转向是正确的。当前主线可以概括为：

```text
TSP 体素 + TBM 表面部件 + 监测响应
        ↓
里程参照下的异质空间实体建模
        ↓
active zone / distance decay / surface-normal compatibility / excavation state
        ↓
几何约束岩–机候选关系图
        ↓
Ac(t), Ic(t) 部件—里程空间交互描述子
        ↓
persistence residual 一致性诊断
```

这条线比原来的 GNN–GRU 主线更稳，原因有三点：

1. **避免小样本深度学习硬伤**：当前有效测试样本有限，难以支撑异构 GNN + GRU 作为主预测模型。
2. **更符合 IJGIS/IJDE 的空间信息逻辑**：图模型服务于空间实体—关系表达，而不是为了追求预测 SOTA。
3. **解释边界更清楚**：描述子是几何加权的 TSP 异常摘要，不是接触力、卡机概率或风险标定。

当前稿件的主要问题已经不是主线，而是：

- 方法公式仍有跨里程可比性问题；
- 结果选择规则需要更透明；
- 个别旧版残留需要清理；
- 结果证据链略薄，需要补充图构建统计和诊断矩阵展示；
- 相关工作还需从 GNN 预测转向 spatial interaction / graph descriptor。

# 一、必须修改项

## 1. 修改 `q_i` 的归一化方式，保证跨里程可比

### 当前问题

当前稿件中 `q_i` 在每个 active snapshot 内部做 min–max 归一化：

```latex
q_i =
\frac{
\max_{\ell\in V_t^r} z(Vp_\ell)-z(Vp_i)
}{
\max_{\ell\in V_t^r} z(Vp_\ell)-\min_{\ell\in V_t^r} z(Vp_\ell)+\epsilon
}.
```

这个定义的问题是：每个里程步 `t` 都会被重新拉伸到 `[0,1]`。这样虽然保留了单个 active zone 内部的相对低速位置，但削弱了不同里程之间 `I_c(t)` 的可比性。

而论文后文正是要分析：

```latex
I_c(t) \leftrightarrow e^{(k)}_{t+h}
```

即沿里程比较部件—里程描述子与响应残差的共变关系。因此 `q_i` 必须采用**固定参照系**归一化，而不能每个里程步单独归一化。

### 修改方案

建议使用训练 active-zone 分布作为固定参照：

```latex
q_i = \operatorname{clip}\left(
\frac{Q_{95}^{train}(Vp)-Vp_i}
{Q_{95}^{train}(Vp)-Q_{5}^{train}(Vp)+\epsilon},
0,1
\right),
```

其中：

- `Q_95^train(Vp)` 和 `Q_5^train(Vp)` 是训练区间 active-zone 中 `Vp` 的第 95 和第 5 百分位；
- `clip(·,0,1)` 保证异常分数位于 `[0,1]`；
- `Vp` 越低，`q_i` 越高；
- 由于参照分布固定，`I_c(t)` 沿里程具有可比性。

### 建议替换文字

将 3.3 中关于 `q_i` 的定义改为：

```text
In the main formulation, qi is defined as a bounded low-velocity anomaly score derived from the training active-zone distribution of P-wave velocity:

qi = clip((Q95^train(Vp)-Vp_i)/(Q95^train(Vp)-Q5^train(Vp)+epsilon), 0, 1).

This fixed training-reference scaling avoids per-chainage min–max normalisation and keeps descriptor values comparable along chainage. Lower Vp values therefore correspond to higher geological anomaly scores under a common reference distribution.
```

### 修改优先级

**最高。** 这是当前稿件最重要的方法公式问题。

## 5. 明确 Figure 4(b) 和 Figure 5 中 fixed pair 的选择规则

### 当前问题

### 修改方案

在 5.3 或 5.4 中加入：

```text
The descriptor–response pairs used in Figure 4(b) and Figure 5 were selected from the main-setting component–response matrix and then held unchanged across all threshold combinations. The reported correlations are exploratory and unadjusted for multiple testing.
```


```text
The fixed pairs were pre-specified according to expected component–response mechanisms before threshold sensitivity analysis.
```

## 8. 说明 `A_c(t)` 受部件采样密度和表面积影响

### 当前问题

你定义：

```latex
A_c(t)=
\sum_{j\in V_c^m}\sum_{i:(i,j)\in E_t^{rm}} w_{ij}^{rm}(t)
```

这是总几何暴露量，会受到：

- 部件节点数量；
- 表面面积；
- 采样密度；
- 候选边数量；
- active zone 几何尺寸；

的影响。

如果直接跨 cutterhead / shield 比较，可能不公平。

### 修改方案 A：增加说明

在 3.3 中 `A_c(t)` 定义后加：

```text
Ac(t) is a total geometric exposure measure and is affected by component surface extent and sampling density. It is therefore mainly used to describe total candidate-relation exposure. When direct cross-component comparison is required, a node-normalised or area-normalised variant can be used.
```

### 修改方案 B：增加归一化版本

可定义：

```latex
\bar A_c(t)=
\frac{1}{|V_c^m|}
\sum_{j\in V_c^m}\sum_{i:(i,j)\in E_t^{rm}} w_{ij}^{rm}(t)
```

并说明：

```text
The normalised exposure \bar Ac(t) represents average geometric exposure per sampled surface node.
```

建议至少采用方案 A；如果你结果中大量使用 `A_c(t)`，建议采用方案 B。

# 五、建议最终贡献表述

最终贡献建议写成三条：

```text
First, this study develops a chainage-referenced rock–TBM spatial entity model that places TSP-derived rock voxels, TBM surface components, and monitoring responses in a common excavation coordinate system.

Second, it defines a geometry-constrained rock–machine candidate relation graph in which candidate relations are restricted by active-zone membership, distance decay, surface-normal compatibility, and excavation-state constraints.

Third, it derives component–chainage spatial interaction descriptors from geometry-weighted TSP anomaly summaries and evaluates their consistency with persistence residuals of TBM monitoring responses.
```

避免写：

```text
GNN–GRU prediction model
forecasting improvement
contact recovery
jamming risk prediction
operational early warning
```

# 八、最终判断

当前稿件主线已经正确，不需要再大改方向。真正要做的是：

1. **把方法公式修严谨**；
2. **把结果选择规则讲透明**；
3. **清理旧版 GNN/ablation 残留**；
4. **补一点图构建统计和结果可视化，让体量更足**；
5. **保持 descriptor 的诊断边界，不要上升为接触力、风险或预警。**

这轮改完后，稿件会更像一篇完整的 **IJGIS/IJDE 空间交互图诊断论文**，而不是小样本深度学习预测论文。当前的正确路线就是：

```text
Spatial entity–relation representation
+ geometry-constrained candidate relation graph
+ component–chainage descriptor
+ persistence-residual consistency diagnosis
```

不要再回退到 GNN–GRU 主线。
