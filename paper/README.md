# Paper Materials

This directory contains manuscript source files and writing support materials for
the IJGIS-oriented paper.

## Layout

```text
paper/
├── ijgis-template/       LaTeX manuscript source and bibliography
├── rewriting_output/     Writing diagnostics, section plans, and revision notes
├── PaperSpine/           Local writing-assistant skill/tool material
├── TBM_surface_patch_paper_writing_and_implementation_plan.md
├── 岩机交互图序列框架_论文提纲与实验方案.md
└── TSP_data_cropped.csv
```

## Manuscript Source

The main LaTeX file is:

```text
paper/ijgis-template/main.tex
```

The manuscript is organized around a GIScience framing: spatial entity modeling,
geometry-constrained relation plausibility, response-supervised interpretation,
and graph-to-surface hotspot mapping for TBM excavation processes.

## Writing Outputs

`rewriting_output/` contains intermediate writing artifacts such as motivation
statements, section blueprints, abstract revisions, and revision audits. These are
working files, not final manuscript sections unless manually integrated into the
LaTeX source.
