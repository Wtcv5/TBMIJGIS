\subsection{Constructed rock--machine interaction graph sequences}
\label{sec:constructed_graphs}
The first result is the constructed spatial relation object itself. For each
case, TSP-derived rock voxels and parameterized TBM surface nodes are mapped to
the same chainage-referenced coordinate system, and the candidate relation set
$E_t^{rm}$ is rebuilt at each excavation step. The graph is therefore not a
conceptual diagram: it is an explicit sequence of screened rock--machine
relations controlled by active-zone membership, distance threshold,
surface-normal compatibility, and excavation state.

Figure~\ref{fig:geometry_edges} illustrates the main construction rule. The
important diagnostic consequence is that each candidate relation is traceable
to a rock voxel, a TBM surface component, a chainage step, a distance, a normal
compatibility value, and a geometry weight $w_{ij}^{rm}(t)$. This traceability
is what monitoring-only curves do not provide.

\subsection{Component--chainage spatial interaction patterns}
\label{sec:component_chainage_patterns}
The main output of the revised framework is the component--chainage descriptor
map built from $A_c(t)$ and $I_c(t)$. In the final result set, this subsection
should report the cutterhead, front-shield, middle-shield, and tail-shield
descriptor values across chainage for each case. The values should be described
as geometry-weighted TSP anomaly summaries over candidate rock--machine
relations, not as contact pressure, contact force, jamming probability, or
calibrated risk.

Figure~\ref{fig:descriptor_evidence} summarises the descriptor evidence chain.
The heatmap panel shows the SJLS component--chainage pattern of $I_c(t)$, the
correlation panel reports the strongest $I_c(t)$--residual association in each
formal case, and the graph-density panel confirms that the result is based on
explicitly constructed candidate rock--machine relations.

\subsection{Association with persistence residuals}
\label{sec:residual_results}
The revised evaluation target is response-residual consistency rather than
absolute response forecasting. For each response variable $k$, the persistence
residual is
\[
e_{t+h}^{(k)}=r_{t+h}^{(k)}-r_t^{(k)}.
\]
The descriptor evidence should therefore report the association between
$I_c(t)$ and $e_{t+h}^{(k)}$ across the evaluated chainage interval. Suitable
summaries include Pearson correlation, Spearman correlation, and overlap
between high-$I_c(t)$ intervals and high-$|e_{t+h}^{(k)}|$ intervals.

Table~\ref{tab:descriptor_residual} reports the strongest descriptor--residual
association in each case. The clearest support is obtained in SJLS, where
cutterhead $I_c(t)$ is strongly associated with the shield-pressure residual
($\rho=-0.907$). BSLL h=3 provides a suggestive front-shield
$I_c(t)$--shield-pressure residual association ($\rho=-0.750$), while BSLL h=1
is weaker and mainly supports construction of the descriptor rather than a
strong residual claim.

\subsection{Robustness and remaining checks}
\label{sec:robustness}
The current evidence supports a bounded descriptor-based claim. SJLS provides
strong descriptor--residual consistency, BSLL h=3 provides suggestive evidence,
and BSLL h=1 remains weak because the test interval is short. Alternative
definitions of $q_i$, spatial detrending, and additional tunnel cases remain
necessary before stronger diagnostic claims are made.
